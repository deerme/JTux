This is the JTux Workstation Programming Reference. This document describes the Java API for Tuxedo/WS provided by JTux Workstation.

The JTux Workstation API

The JTux Workstation API is a direct mapping of the Tuxedo's native C API to Java. The methods of the jtux.atmi.ATMI class, for example, are the Java versions of Tuxedo's native ATMI functions. Similary, the methods of the jtux.fml32.FML32 class are the Java versions of Tuxedo's native FML32 functions. The different Tuxedo data structures are also mapped to Java in a straightforward manner, as shown by, for example, the jtux.atmi.TPINIT and jtux.atmi.TPQCTL class definitions.

Differences with Tuxedo's Native C API

Although the JTux Workstation API is mostly similar to Tuxedo's native C API, there are also some differences. In particular, the following topics deserve special attention.

Error and event reporting
The JTux Workstation API reports errors and events by throwing exceptions. The methods of the jtux.atmi.ATMI class, for example, throw exceptions of type {@link jtux.atmi.TPException} and the methods of the jtux.fml32.FML32 class throw exceptions of type {@link jtux.fml32.FException} to signal the occurrence of errors and events.
Typed buffer handling
The JTux Workstation API uses instances of the java.nio.ByteBuffer class to represent Tuxedo typed buffers. Special care should be taken that these buffers are properly cleaned up as they are not managed by the Java garbage collector.
Passing parameters by reference
The JTux Workstation API uses holder classes to allow method parameters to be passed by reference. These holder classes are defined in the {@link jtux.holders} package.

Sample Applications

The following sample applications are available to get you started developing your own Tuxedo/WS clients in Java. For instructions on how to compile and run these samples see the JTuxWS.jar manual page.

Name Purpose
SimpleClient Shows the basics of calling a Tuxedo service from Java.
StringUtils Utility methods for dealing with Tuxedo STRING buffers in Java.
SimpleAsyncClient Shows how to call a Tuxedo service asynchronously.
DownloadClient Shows how to write a conversational client.
SimpleEnqueuer Shows how to enqueue a message in Tuxedo/Q queue.
SimpleDequeuer Shows how to dequeue a message from a Tuxedo /Q queue.
UnsolClient Shows how to handle unsolicited messages.
SimpleSubscriber Shows how to subscribe to events posted through the Tuxedo event broker.
SimpleFML32Client Shows how to read and write FML32 buffers.
