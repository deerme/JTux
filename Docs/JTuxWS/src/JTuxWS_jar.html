JTuxWS.jar

JTuxWS.jar is the Java class library containing the Java API for Tuxedo/WS described in the JTux Workstation Programming Reference. The JTuxWS.jar file is located in the lib subdirectory of the JTux Workstation directory.

Compiling JTux Workstation clients using JTuxWS.jar

To compile a JTux Workstation client using JTuxWS.jar, simply add the JTuxWS.jar file to the classpath of the Java compiler. For example, the command for compiling the SimpleClient class written in Java looks like this: 

javac -classpath JTUXWSDIR/lib/JTuxWS.jar -d dir SimpleClient.java
where JTUXWSDIR is the JTux Workstation directory and dir is the output directory in which to write the SimpleClient.class file.

Running JTux Workstation clients using JTuxWS.jar

To run a JTux Workstation client using JTuxWS.jar, the JTuxWS.jar class library must be present in the class path of the Java application launcher. Furthermore, at runtime, JTuxWS.jar uses a native code library called JTuxWSInterop and the directory containing this library must be passed to the Java application launcher using the java.library.path system property. On Unix, this is the lib and on Windows the bin subdirectory of the JTux Workstation directory. Finally, to run a JTux Workstation client using a 64-bit version of JTux Workstation, the -d64 command line option must be passed to the Java application launcher as well.

For example, the following command runs the SimpleClient sample application using a 64-bit version of JTux Workstation on Unix: 

java -d64 -classpath JTUXWSDIR/lib/JTuxWS.jar:dir -Djava.library.path=JTUXWSDIR/lib SimpleClient "Hello, World!"
and the following command runs the SimpleClient sample application using a 32-bit version of JTux Workstation on Windows: 
java -classpath JTUXWSXDIR\lib\JTuxWS.jar;dir -Djava.library.path=JTUXWSDIR\bin SimpleClient "Hello, World!"
In these examples, JTUXWSDIR is the JTux Workstation directory and dir is the directory containing the SimpleClient.class file.

These examples assume that the Tuxedo system environment variables are set as described in the Tuxedo installation guide (Chapter 6 "Performing Post-Installation Tasks", sections "Setting Environment variables on a Windows System" and "Setting Environment variables on a UNIX system") and the Tuxedo setup guide (Chapter 11, "Setting up Workstation Clients", section "Defining Workstation Clients").
