JTux Workstation

This is the technical documentation set for JTux Workstation version 1.1.

Target Audience

This technical documentation set is intended for Tuxedo developers who want to develop Tuxedo/WS clients in Java using JTux Workstation. Readers are expected to know how to:

  • Install and configure Tuxedo/WS.
  • Develop Tuxedo/WS clients in C.
  • Install a Java SDK.
  • Work with Java class libraries and the Java class path.
  • Develop applications in Java.

Getting started

To get started developing Tuxedo/WS clients in Java:

  1. Install JTux Workstation as described in the JTux Workstation Installation Guide.
  2. Get familiar with the JTux Workstation API by studying the JTux Workstation Programming Reference.
  3. Develop and run your own Java based Tuxedo/WS clients using the JTuxWS.jar class library.

Documentation Index

The JTux Workstation documentation set consists of the following documents:

  • JTux Workstation Installation Guide
    Describes how to install JTux Workstation on a client machine in your local Tuxedo environment.
  • JTux Workstation Programming Reference
    Describes the JTux Workstation API and illustrates how to use it by means of a number of sample Tuxedo/WS clients written in Java.
  • JTuxWS.jar Manual Page
    Describes how to compile and run Java based Tuxedo/WS clients using the JTuxWS.jar class library.
