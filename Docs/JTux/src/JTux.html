JTux

This is the technical documentation set for JTux version 1.1.

Target Audience

This technical documentation set is intended for Tuxedo developers who want to develop Tuxedo clients and servers in Java using JTux. Readers are expected to know how to:

  • Install and configure Tuxedo.
  • Develop Tuxedo clients and servers in C.
  • Install a Java SDK.
  • Work with Java class libraries and the Java class path.
  • Develop applications in Java.

Getting started

To get started developing Tuxedo clients and servers in Java:

  1. Install JTux as described in the JTux Installation Guide
  2. Get familiar with the JTux API by studying the JTux Programming Reference.
  3. Develop and run your own Java based Tuxedo clients and servers using the JTux.jar class library and the JServer executable.

Documentation Index

The JTux documentation set consists of the following documents:

  • JTux Installation Guide
    Describes how to install JTux on a server machine in your local Tuxedo environment.
  • JTux Programming Reference
    Describes the JTux API and illustrates how to use it by means of a number of sample Tuxedo clients and servers written in Java.
  • JTux.jar Manual Page
    Describes how to compile Tuxedo clients and servers written in Java and how to run Java based Tuxedo clients using the JTux.jar class library.
  • JServer Manual Page
    Describes how to run Java based Tuxedo servers using the JServer executable.
  • JTMS Manual Page
    Describes how to configure a JTMS server group to perform JDBC and JMS operations as part of distributed transactions managed by Tuxedo.
